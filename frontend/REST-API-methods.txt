REST API requesty pro frontend
v závorkách příklad URL API

Menu
* GET položky v denní nabídce (/api/menu-items/daily)
	** vracet podle typu položky
	** id položky
	** název položky
	** cena
* GET položky stálého menu (/api/menu-items/permanent)
	** vracet podle typu položky
	** id položky
	** název položky
	** cena
* GET nápoje
	** vracet podle typu položky
	** id položky
	** název položky
	** cena

Objednávky
* GET všechny objednávky (/api/orders)
	** id objednávky
	** stůl
	** objednané položky (seznam názvů a jejich tipů, např. pivo-nápoj nebo svíčková-denní-nabídka)
	** datum
	** čas vytvoření objednávky
	** čas vyřízení objednávky
* GET objednávky filtrované podle datumu (/api/orders?date=20-11-2018)
	** id objednávky
	** stůl
	** objednané položky (seznam názvů a jejich tipů, např. pivo-nápoj nebo svíčková-denní-nabídka)
	** datum
	** čas vytvoření objednávky
	** čas vyřízení objednávky
* GET současné nevyřízené objednávky (/api/orders/pending)
	** id objednávky
	** stůl
	** objednané položky (seznam názvů a jejich tipů, např. pivo-nápoj nebo svíčková-denní-nabídka)
	** datum a čas vytvoření objednávky
	** datum a čas vyřízení objednávky
* GET // jedna objednávka s více detaily (/api/order/21)
	** id objednávky
	** stůl
	** objednané položky (seznam názvů a jejich tipů, např. pivo-nápoj nebo svíčková-denní-nabídka)
	** datum a čas vytvoření objednávky
	** datum a čas vyřízení objednávky
* POST Objednání položky (/api/menu-items/order?item_id=6)
	** přidání záznamu do databáze asi podle ID položky menu
* PUT Označení objednávky jako "připravená" (/api/order/3/ready)
	** 3 je ID objednávky
* PUT Označení objednávky jako "vyřízená" (/api/order/3/done)
	** 3 je ID objednávky
